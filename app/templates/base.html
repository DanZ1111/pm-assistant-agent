Quick Start

Creating a project

  1. Click New Project in the navbar
  2. Enter a Project Name (only required field)
  3. Fill in Brand, PM, Engineer, Factory, Target Cost, MSRP, Launch Date, and Thesis if available
  4. Select prototype rounds: Single (8 phases) or Double (10 phases)
  5. Click Create Project

Tip: Use AI Intake if you have product notes or a spec sheet — AI can extract all fields automatically.

What does "Needs Info" mean?

The project is missing one or more critical fields: brand, PM, engineer, factory, target cost, MSRP, launch date, or Product Thesis. It doesn't block the project — it's a reminder that the project isn't fully filled in yet.

What does "Delayed" mean?

At least one phase has a planned end date in the past and isn't marked Done or Skipped. Delay is calculated automatically — you can't manually set it. To clear a delay: mark the phase Done, or update its planned end date.

What should a Product Thesis include?

  • Why this product exists
  • Who it is for and what problem it solves
  • What makes it different from existing options
  • Why it belongs to this brand
  • Target price logic
  • Main risks or unknowns

Minimum 80 characters — shorter entries are treated as incomplete.

How to use AI Intake

Option A — Paste text: Go to AI Intake → paste notes/emails/messages → Extract Fields → review and edit → Confirm & Create.

Option B — Upload file: Go to AI Intake → upload a PDF or image → Analyze File → review proposed fields and AI summary → Confirm & Create. The file is automatically attached to the project.

File categories

Rendering · Reference · Quotation · Factory Feedback · Thesis · Packaging · Other. Use the category tabs in the gallery to filter.
